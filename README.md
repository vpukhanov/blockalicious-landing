# Blockalicious Landing Page

Landing website for [Blockalicious](https://github.com/vpukhanov/blockalicious), a privacy-first website blocker for Safari.

## About Blockalicious

Blockalicious is a Safari extension that blocks distracting websites through on-device blocking using Safari's native content blocker API. No servers, no analytics, no data collection.

- **Download**: [App Store](https://apps.apple.com/app/blockalicious/id1608665824)
- **Source**: [GitHub](https://github.com/vpukhanov/blockalicious)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
